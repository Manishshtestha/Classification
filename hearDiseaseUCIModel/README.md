# Heart Disease Presence Prediction

This project includes building classification Model for predicting the presence of heat disease. It uses heart disease dataset from kaggle which has 14 attributes including target. The "target" field refers to the presence of heart disease in the patient. It is integer valued 0 = no disease and 1 = disease.
